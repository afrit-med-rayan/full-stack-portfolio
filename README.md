# Mohamed Rayan Afrit — Portfolio

A bold, modern, and high-end personal portfolio built with **Next.js 14**, **Tailwind CSS**, and **TypeScript**. Designed to showcase expertise in Full-Stack Development, AI Engineering, and Data Science.

![Portfolio Preview](/assets/profile.jpg)

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **language:** TypeScript
- **Animations:** Framer Motion & CSS Animations
- **Icons:** Lucide React
- **Typography:** Syne (Headings) + Inter (Body)
- **Deployment:** Vercel

## ✨ Key Features

- **Performance-First:** Built using Next.js 14 with optimized image loading and server components.
- **Modern Design:** High-contrast dark/light theme sections with a premium "dev studio" aesthetic.
- **Interactive Projects:** Featured project spotlight with explicit "Demo" and "Code" links.
- **Oracle Certifications:** Clickable verification badges for Cloud Professional credentials.
- **Responsive:** Fully optimized for all screen sizes, from mobile to ultra-wide desktops.
- **Contact Integration:** Easy-to-reach contact section with direct email and social links.

## 📁 Project Structure

```bash
├── public/
│   └── assets/          # Images, CV, and static assets
├── src/
│   ├── app/             # Next.js App Router (Layout & Page)
│   └── components/      # Modular React components (Hero, About, Projects, etc.)
├── tailwind.config.js   # Custom design tokens & theme
└── tsconfig.json        # TypeScript configuration
```

## 🛠️ Personal Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/afrit-med-rayan/full-stack-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
